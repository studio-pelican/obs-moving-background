# 実装計画：動くライブ配信背景（OBSブラウザソース用）

本ドキュメントでは、OBSブラウザソースとして使用する「動くライブ配信背景」のビジュアルおよびアニメーションの詳細仕様を定義します。

## 概要
* **目的**: OBSのブラウザソースとして機能する、美しく軽量な動的背景
* **構成**: HTML/CSS/Vanilla JSのみ（外部依存ライブラリなし）
* **基本機能**: JST時計の常時表示、URLパラメータによるテーマ（Game/Talk）の切り替え、配信者名の表示

## 1. ビジュアル設計およびアニメーション仕様

### 1.1. Talkテーマ（雑談配信用）
**コンセプト**: 落ち着きのある、柔らかくリラックスできる空間。

* **カラースキーム**: 夜空やマジックアワーを思わせる、なだらかなグラデーション（例：ディープパープルからソフトピンクやピーチ）。`?custom`パラメータで基準色を変更可能です。
* **背景アニメーション (CSS)**: `linear-gradient` の `background-position` をCSS keyframesで非常にゆっくりとアニメーションさせ、波打つような・色が移り変わるような静かな動きを実現します。
* **パーティクル (Canvas)**:
    * 画面下から上へ複数種類の透明度を持つ円形（星やシャボン玉、光の粒のイメージ）がゆっくりと浮遊します。
    * 動きはスローペースで、サイズや出現位置をランダムに散らします。画面上端に到達したらフェードアウトし、下から再出現します。
* **フォント・レイアウト**: 丸みのある読みやすいサンセリフフォント。時計は右上に配置。全体の負荷は非常に低めです。

### 1.2. Gameテーマ（ゲーム配信用）
**コンセプト**: サイバー感、ネオンパレット、ピクセルアートを取り入れたゲーマー向けデザイン。

* **カラースキーム**: ダークな背景に鮮やかなネオンカラー（例：シアンブルーとマゼンタ。`?custom`パラメータでネオンカラーを変更可能です）。
* **背景アニメーション (CSS)**: 遠近感のあるグリッド線（`linear-gradient`で擬似的に描画）を奥から手前へスクロールさせることで、サイバー空間を進んでいるようなスピード感を演出します。
* **パーティクル (Canvas)**:
    * 正方形（ピクセル風）のパーティクルがランダムに点滅しながら漂う、または放射状に軽く飛ぶエフェクトを描画します。
    * 中央背景側に薄い不透明度でゲームコントローラーのシルエット（SVGパスをCanvas描画、またはCSSで配置）を配置し、ゆっくりと上下に浮遊させます。
* **フォント・レイアウト**: デジタル時計を意識したmonospace系、または角張ったクールなフォント。時計は右上に配置。負荷を抑えるために同時に描画する要素数を制限し、60fpsを維持します。

## 2. 共通・技術仕様

* **レスポンシブ・OBS最適化**: フルHD（1920x1080）を基準に作成。`body { background: transparent; overflow: hidden; margin: 0; }` を適用し、OBSのカスタムCSS設定などで配信画面と重ね合わせやすいようにします。
* **URLパラメータ**:
    * `?theme=talk` または `?theme=game`（未指定時は `talk`）
    * `?name=配信者名`（指定時は画面中央などにスタイリッシュに表示）
    * `?custom=%23FF0000`（指定時はテーマのメインアクセントカラーを上書き）
* **時計**: `Date.toLocaleString('ja-JP', { timeZone: 'Asia/Tokyo' })` を使い、毎秒JSTのデジタル時計（`HH:MM:SS`）を更新します。
